                                                                  ABSTRACT

                                         RGB and depth (D) images vary significantly in different
                                         scenes. In this paper, we propose an Attention Complemen-
                                         tary Network (ACNet) that selectively gathers features from
                                         RGB and depth branches. The main contributions lie in the
